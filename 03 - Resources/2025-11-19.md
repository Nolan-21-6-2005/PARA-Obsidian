# Artificial Intelligence
Các lĩnh vực chính: 
- Thị giác máy tính(Computer vision): 
	- Mạng Neural: tích chập
	- Biểu diễn tri thức: Scene graph, Bayes network
	- Cảm biến: Camera, lidar, ...
	- Kiểu tác nhân: 
		-  _Reactive agent_ (tác nhân phản ứng nhanh, ví dụ robot tránh vật cản).
		- _Model-based agent_ (tác nhân có mô hình trạng thái của môi trường).

- Xử lý ngôn ngữ tự nhiên (Natural processing language): 
	- Mạng Neural: hồi quy, tranformers
	- BIểu diễn tri thức: Mạng ngữ nghĩa.
	- Kiểu tác nhân:
		- _Goal-based agent_ (trả lời câu hỏi, sinh văn bản để đạt mục tiêu).
	    - _Learning agent_ (tự học từ dữ liệu văn bản).

- Hệ chuyên gia (Expert System):
	- Mạng Neural: hồi quy.
	- Biểu diễn tri thức:
	- Cảm biến: Camera(thị giác máy tính), văn bản(NLP).
	- Kiểu tác nhân:
		- _Knowledge-based agent_ (tác nhân suy luận dựa trên tri thức và luật).
# Data structure & algorithms
1. Linked list: danh sách liên kết.
  - Là nền tảng của các cấu trúc dữ liệu trừu tượng: stack, queue.
  - Ứng dụng: Danh sách tiến trình máy tính, danh sách gói tin mạng được truyền đi.

3. Hash table + hash function:
  - Cấu trúc tra cứu nhanh O(1).
  - Ứng dụng: bảng băm cho từ điển, routing table, bộ nhớ đệm (cache), quản lý phiên đăng nhập.

  - FIFO.
  - Ứng dụng: lập lịch CPU, hàng đợi gói tin mạng, hệ thống đặt vé, message queue.
