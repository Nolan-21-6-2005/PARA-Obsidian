# Định nghĩa
Được định nghĩa bằng tỉ số vi phân:
$$
L =\displaystyle \lim_{ n \to 0 } \frac{f(x + h) - f(x)}{h} 
$$
Giả sử $f(x) = x^2$ => $f'(x) = 2x$ vì:
Đặt:
$$
G(x)=\frac{(x+h)^2 - x^2}{h}
$$
$$
G(x) = \frac{x^2+2xh+h^2 - x^2}{h}
$$
$$
G(x) = \frac{2xh+h^2}{h}
$$
$$
L = \displaystyle \lim_{ n \to 0 } G(x)
$$
$$
L = \displaystyle \lim_{ n \to 0 } \frac{2xh+h^2}{h}
$$
$$L = 2x$$
