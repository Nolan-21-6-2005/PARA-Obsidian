Bộ 5 bài tập tự luyện (ẩn đáp án):

---

### **Bài 1**

```c
for (i = 1; i <= n; i++)
    for (j = 1; j <= n * n; j++)
        x++;
```

→ Tính ( T(n) ).

---

### **Bài 2**

```c
for (i = 1; i <= n; i++)
    for (j = 1; j <= i; j++)
        for (k = 1; k <= j; k++)
            sum++;
```

→ Tính ( T(n) ).

---

### **Bài 3**

```c
i = n;
while (i > 0) {
    i = i / 2;
}
```

→ Tính ( T(n) ).

---

### **Bài 4**

```c
for (i = 0; i < n; i++)
    for (j = 0; j < 100; j++)
        work++;
```

→ Tính ( T(n) ).

---

### **Bài 5**

```c
for (i = 1; i <= n; i++)
    for (j = 1; j <= n; j *= 2)
        count++;
```

→ Tính ( T(n) ).
